# Road4AI May Roadmap: The Workload Phase

## Strategy: From Infrastructure to Impact
In April, we built the "Bridge" (Hermes + Gemini CLI + CrewAI). In May, we cross it. We will demonstrate real-world workloads where agents solve complex problems using persistent context.

## Content Calendar (Target: LI / TT / IG / FB)

- **Goal:** Show how multiple agents "share" a brain via Hermes.
