## Post templates

### X thread template

1. Start with a strong claim.
2. Name the enemy or broken default.
3. Show the alternative Road4AI proposes.
4. Add one concrete artifact from the repo.
5. End with a repo CTA.

### LinkedIn template

1. Open with a real friction point.
2. Reframe it as a systems problem.
3. Explain the Road4AI position.
4. Give one practical example.
5. End with a direct CTA.

### Blog template

1. Define the problem.
2. Explain the common bad default.
3. Present the Road4AI approach.
4. Show steps, commands, or examples.
5. Close with the next action.
