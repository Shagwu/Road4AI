# Road4AI Content Strategy

## Goal
Establish Road4AI as the go-to resource for developers building real-world AI agent systems.

## Content Pillars
1. **Agent Coordination Patterns**: How agents work together (HNSW, state files, protocols).
2. **Developer Tooling**: Deep dives into MCP, Blotato, and custom CLI tools.
3. **Engineering Rigor**: TDD for agents, validation cycles, and safety.
4. **Behind the Scenes**: Building Road4AI itself.

## Target Platforms
- **LinkedIn**: Long-form technical analysis, architectural diagrams, and engineering leadership.
- Twitter/X: Quick tips, "ship logs," and interactive code snippets. **Hard limit: 280 characters per post (including threads).**

## Cadence
- **Deep Dive**: 1x Weekly (LinkedIn).
- **Ship Log**: 3x Weekly (Twitter/X).
