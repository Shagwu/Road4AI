Spent an hour last week trying to get a 14B local model to do something a five-line script could've done in ten seconds.

That's the trap. Once a multi-agent pipeline is running, everything starts looking like an agent problem.

The actual skill isn't building bigger pipelines. It's knowing when a tool needs real care, adversarial review, confidence checks, a human on the diff, and when simple deterministic code is not just enough but better, because it never hallucinates.

Every task handed to a model instead of a script is attention not spent on the decisions that need a human. Which model to trust unsupervised. When a suspiciously clean confidence result should worry you instead of relieve you.

Good judgement isn't using AI everywhere. It's knowing exactly where not to.
