Tomorrow the numbers go public. Here's the one thing I keep coming back to.

Not the score. Not the benchmark. Not the +22%.

It's the fact that I almost published the wrong number.

Last week, the reveal posts said 0.871. That's the deterministic score. Different execution path. Not the live result. The real number is 0.788. Same model, same cases, same Ollama instance running on my laptop.

I caught it because I traced every claim back to a source file. The code was right. The narrative was lying.

That's the part nobody talks about when you build in public. The hardest part isn't getting the system to work. It's making sure what you say about the system matches what the system actually did.

Tomorrow we find out if the work holds up.
