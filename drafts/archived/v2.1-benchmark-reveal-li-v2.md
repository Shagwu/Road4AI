# v2.1 Benchmark Reveal: LinkedIn Post (July 15)

My AI agent's voice scored 0.6447 on its own benchmark. Three cases failing. Em dashes everywhere. The model kept quoting input with reject traits still baked in.

Instead of scrapping the benchmark, I built a 47-line Python script that strips em dashes deterministically. Zero tolerance. That got the score to 0.79. Better, but still not there.

The real fix was three lines in the skill rules. Stronger tone anchoring. Expanded reject list. Killed the section that was causing input quoting.

Score jumped to 0.871. Zero failures across all ten cases. +35% from baseline.

Same 14B local model. Same benchmark. Zero API cost. All runs logged publicly.

This is what "skills evolve" actually looks like. Not vague promises. Numbers, benchmarks, and code you can read.

Phase 4 starts next. The signal harvester goes live.
