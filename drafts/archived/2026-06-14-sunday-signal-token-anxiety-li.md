# LinkedIn Draft: The Sunday Signal

I changed how I capture messy ideas.

The old pattern was obvious:

Wait until the thought is useful.
Open the AI tool.
Ask it to help shape the idea.

That works until the thought shows up before the tool is ready.

Or the request window feels scarce.

Or the model takes too long.

Or you start negotiating with yourself:

"Is this worth using a message on?"

That question is a trap.

So I borrowed the same pattern from my Sunday dump.

One raw capture file.
No formatting pressure.
No performance.
No attempt to make it sound like content.

Just sections:

Struggles.
Wins.
Consumed.
Questions.
Joy moments.

Then three Sunday questions:

What frustrated me most this week?
What worked that surprised me?
What am I still thinking about?

The important part is not the template.

The important part is separating capture from synthesis.

Capture is for protecting signal.

Synthesis is for shaping signal.

When those two collapse into the same step, the tool starts controlling what gets remembered.

That is how token anxiety turns into lost ideas.

The fix is boring:

Write the messy sentence first.

Let the model help later.
