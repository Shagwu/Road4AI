I saw a tweet today describing a "permanent memory system" for AI agents.

The setup: OMI takes notes in the background. Obsidian stores them as text files. Every agent reads the vault on startup.

It works. But it's a vault, not a brain.

Here's the difference:

A vault stores everything. A brain decides what matters.

When your agent reads 500 text files every morning, it doesn't know which ones are still relevant, which ones are outdated, or which ones contradict each other. It just loads everything and hopes for the best.

That's how you get an agent that "remembers" your Q1 goals in Q3. Or one that cites a client brief from six months ago as if it's current.

Hermes doesn't do that. It scores relevance. It expires old context. It monitors drift between what it thinks and what's actually true.

The vault is step one. Most people stop there.

The brain is what happens next.
