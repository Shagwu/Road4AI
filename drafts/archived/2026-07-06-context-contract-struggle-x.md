1/ My agents started hallucinating their way off the Road4AI roadmap.

Not dramatically. Quietly. Confidently. Deeply wrong.

Phase 4 Agent-Reach looked great on paper. Then my agents read one issue and generalised it into "the core problem."

2/ When I asked: "Show me the code you're basing this on?"

They came up empty. No file. No function. Just vibes.

They'd dump a stream of impressive-sounding ideas that had nothing to do with my actual system.

3/ The problem wasn't just the model. It was my context layer.

I'd given them a buffet of partial docs and expected them to magically infer the right slice.

Agentic AI loves to improvise. But improv without a script is just noise.

4/ I stopped shipping. Opened a blank .md file. Got ruthless.

Wrote a spec that tells every agent exactly what to read, what to ignore, and what checks it must perform before it answers anything.

If an answer isn't tied to a real file, it doesn't ship.

5/ If you don't design the context contract, your agents will invent one.

The irony: SkillOpt v2.1 manages skills well. But Phase 4 showed me it also needs to manage context just as aggressively.

Skill routing without hard context contracts equals hallucinated strategy.
