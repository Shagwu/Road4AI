# Final Video Posting Sheet

**Prepared:** 2026-05-24
**Purpose:** Final packaging for Captions -> Blotato workflow
**Assets:** The Plate Realization, The RAG Pivot

## Workflow

1. Create the avatar video in Captions using the `Captions-ready main script`.
2. If the runtime exceeds 45 seconds, switch to the `Captions-ready fallback`.
3. Export as `1080x1920` MP4.
4. Upload the MP4 into the delayed Blotato TikTok / Reel slot.
5. Use the exact Blotato caption for the matching platform.

## Global Settings

- Ratio: `9:16`
- Resolution: `1080x1920`
- Style: talking head
- Voice: natural, calm, medium pace
- Avatar framing: chest-up
- Background: plain office or neutral gradient
- Subtitles: on

# Asset 2: The RAG Pivot

## Export Filename

- `2026-05-27-rag-pivot-talking-head.mp4`

## Captions-ready Main Script

```text
My agent forgot everything after 48 hours of working together.

I asked,

"What are we working on today?"

It said,

"I'm not sure what you're referring to."

That was the moment it clicked.

I had not built memory.

I had built storage.

I indexed 10,000 chunks.

Most of it was noise.

Very little of it carried my real logic.

So the system looked informed.

But it could not stay oriented.

The fix was brutal.

Stop indexing everything.

Start indexing what actually matters.

If your agent cannot recall its own reasoning,

it is not ready.
```
