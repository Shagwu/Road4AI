# Ask Docs Skill

This skill allows the agent to query the official CrewAI documentation.

## Overview
It uses a structured workflow to fetch the documentation index (`llms.txt`), identify relevant pages, and retrieve their content to provide accurate, cited answers.

## Usage
Activate this skill when you need the latest information on:
- API Reference and parameter details
- Advanced configuration options
- Troubleshooting and error resolution
- New features not yet covered in local skills
