# Installed Skills For Road4AI

These skills were selected from the bundled `marketing-skills` repository because they fit Road4AI's current needs as an open-source, developer-facing project:

- `product-marketing-context`
- `launch-strategy`
- `content-strategy`
- `social-content`
- `copywriting`
- `ai-seo`

Why these:

- `product-marketing-context` gives the other skills a shared source of truth.
- `launch-strategy` fits Road4AI's manifesto-driven launches and feature drops.
- `content-strategy` supports blog, tutorial, and educational roadmap planning.
- `social-content` supports distribution on X, LinkedIn, and community channels.
- `copywriting` helps tighten landing, README, and site messaging.
- `ai-seo` fits discovery through AI answers, search summaries, and citations.

Installed from:

- `marketing-skills/skills/*`

Supporting context file:

- `.agents/product-marketing-context.md`
