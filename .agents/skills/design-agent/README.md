# Design Agent Skill

Expert guidance on designing and configuring CrewAI agents.

## Overview
This skill implements the **Role-Goal-Backstory** framework to ensure agents are specific, outcome-focused, and qualified. It also covers advanced configuration including execution control, tool assignment, and guardrails.

## Key Principles
- **80/20 Rule:** 80% effort on task design, 20% on agent design.
- **Specialization:** Always prefer specialist agents over generalists.
- **YAML Configuration:** Recommended for clean separation of configuration and code.
